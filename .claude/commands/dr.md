---
description: Draft a decision record for a BINDING or architectural change
argument-hint: "[decision summary]"
---
Delegate to the scribe agent to draft the next decision record as a new file under `docs/decisions/` (one record per file, `DR-NNN-<slug>.md`) for the decision in $ARGUMENTS. The scribe numbers it from the existing `docs/decisions/DR-*.md` files, writes it in the house shape (including the strongest counterargument raised), and presents it PROPOSED for acceptance. On acceptance the record file is written and the §20 decision-record index in `docs/rezidnt-architecture.md` is updated (new row + next-number bump, plus any "amended by" pointer on superseded plan sections). Nothing is marked ACCEPTED without explicit owner sign-off.
